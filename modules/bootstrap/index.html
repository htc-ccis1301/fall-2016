Module: Introduction to Bootstrap

An introduction to using the Bootstrap framework to aid in web site design and development.

Learning Outcomes

Use Bootstrap Framework

Build a website that uses the Bootstrap framework to build a responsive, mobile first website.

  • Use and customize the Bootstrap Navbar
  • Use the customize a jumbotron
  • Use a Bootstrap Grid for layout
  • Use Bootstrap CSS classes for image and text formatting

Readings

Bootstrap Introduction

An introduction to using the Bootstrap framework to design and develop web sites.

Bootstrap CSS

An introduction to some of the CSS provided in the Bootstrap framework.

Bootstrap Grid

An introduction to using the Bootstrap grid framework to layout a responsive web site.

Bootstrap Componenets

An introduction to some of the user interface (UI) components provided in the Bootstrap framework.
