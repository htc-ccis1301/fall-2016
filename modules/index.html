Modules Topics covered in this class.

This page presents the “modules”, or the topics that are covered in this course.

Click on the tile associated with a module to go to a page containing that module’s contents.

1. Introduction to Web Development

An overview of web standards and architecture and an introduction to web development.

2. HTML Basics

An introduction to HTML syntax, structure, and common tags.

3. CSS Basics

This module is an introduction to Cascading Style Sheets (CSS).

4. Graphics and Text Styling

This module focuses on adding interest to your web pages through the use of graphics and text styling.

5. CSS Box Model & More

An introduction to the CSS box model and additional CSS features to add color to your pages.

6. Page Layout

This module focuses on layout and positioning of elements on the page.

7. Introduction to Web Design

An introduction to basic elements of web site design.

8. Responsive Web Design

An introduction to the technique of responsive web design and CSS media queries.

9. Building & Styling Tables

10. Web Forms

11. Media & Interactivity

In this module we’ll learn to use audio, video and interactive effects to add interest to our web pages.

Coming soon...

12. Web Publishing & Testing

This module discusses web publishing and testing for both usability and accessibility.

Coming soon...

13. Case Study: Pacific Trails Resort

This is a case study project, which is set up to simulate a real world project. In it, you will develop a web site for a business customer. Unlike other modules, the work for this module is distributed across the semester. Due dates for each part can be found on the schedule.

14. Introduction to Bootstrap

An introduction to using the Bootstrap framework to aid in web site design and development.

15. Final Project & Wrap-up

This module wraps up the course with a review of key topics and a final project.
